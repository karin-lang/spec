## ID

命名規則

- `a-z` `A-Z` `0-9` `_` で構成される
- `0-9` 以外で開始する
- 未使用変数を示すには `_` で開始する

## キーワード

|キーワード|役割|
|:-|:-|
|`elif`|elif 節|
|`else`|else 節|
|`fn`|関数定義|
|`for`|反復式|
|`if`|条件式|
|`in`|繰り返し範囲|
|`let`|変数束縛|
|`mut`|可変化|
|`pub`|アイテム公開|
|`ref`|参照|
|`ret`|値返却|
|`struct`|構造型定義|
|`none`|none 型, none 値|
