# Karin

Karin は Rust 言語に影響された非純粋関数型プログラミング言語である。

安全性と学習容易性を適度に両立することを理念としている。

初期段階では Web アプリケーションへの組み込みを想定して JavaScript へのトランスパイラを持つ。
