# 特性体

概ね Rust 言語のトレイトと同様。Java 言語等の継承に近いが、継承にあたる実装は一方向かつ一階層に制限される。特性体は構造体と列挙体に対して実装できる。

※「実装」は用語として不適切かも

```rb
# 特性体定義
trait T {
    # メソッド
    fn f();
}

# 構造体に対する特性体の実装
struct S {
    fn normal_f() {}

    # 特性体 T の実装
    impl T {
        fn f() {}
    }
}

# 列挙体に対する特性体の実装
enum E {
    Variant;

    # 特性体 T の実装
    impl T {
        fn f() {}
    }
}
```
