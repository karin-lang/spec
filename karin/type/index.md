# データ型

## 基本型

|名称|キーワード|
|:-|:-|
|none 型|`none`|
|論理型|`bool`|
|符号付き整数型|`i8` `i16` `i32` `i64`|
|符号なし整数型|`u8` `u16` `u32` `u64`|
|浮動小数点数型|`f32` `f64`|
|文字型|`char`|
|文字列型|`str`|

## 型推論

```rb
# 型伝播 (i -> j)
let i = 0
let j = i

# 型伝播 (j -> i)
let i = 0
let j usize = i
```

## 型検査

型の不一致が存在する場合に型エラーを送出する。

エラー例）`i32 -> str` `num -> str`
